Plan a Mastodon upgrade across all of its dependencies in one place. Pick your current and
target Mastodon versions, tell the advisor which backends you run and the versions you have,
and it will flag anything you need to change — and in what order.

:::tip
Choose the **cache**, **image processor**, and **search** backend you actually run. Search
is optional; leave it set to *Not used* if you don't run full-text search.
:::
