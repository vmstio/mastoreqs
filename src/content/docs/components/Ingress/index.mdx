Mastodon requires a load balancer, ingress or reverse proxy to properly terminate TLS/SSL connections
between users and the Mastodon web interface and API.

While there is no specific technical requirement or limitation on what software can be used, Nginx is commonly used for this purpose.
Mastodon has no version-specific requirement for ingress software.

:::note
Regardless of the ingress software that is used, you should keep it up to date to prevent exploitation of known security vulnerabilities.
:::
