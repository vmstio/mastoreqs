Mastodon requires an installation of FFmpeg for video processing, and administrators must
ensure a supported FFmpeg version is present and configured.

:::tip
Mastodon versions run across the top of the matrix; component versions run down the left
side. Click any Mastodon version to focus that column.
:::

<SupportMatrix component="ffmpeg" />

References: [FFmpeg EOL Dates](https://endoflife.date/ffmpeg)

```bash title="Terminal"
# Check running version
ffmpeg -version
```

:::note
The official Mastodon [container image](https://hub.docker.com/u/tootsuite) since version 4.4
will always use the highest possible FFmpeg version for that release.
:::
