Mastodon requires an installation of Postgres as the primary SQL database, and administrators
must ensure a supported Postgres version is present and configured.

:::tip
Mastodon versions run across the top of the matrix; component versions run down the left
side. Click any Mastodon version to focus that column.
:::

<SupportMatrix component="postgres" />

References: [Postgres Versioning](https://www.postgresql.org/support/versioning/)

```bash title="Terminal"
# Check running version
psql --version
```

:::danger
Always use caution when moving between major versions of Postgres (ex: 15.x to 16.x) as
failure to adequately plan this migration may result in data-loss!
:::
