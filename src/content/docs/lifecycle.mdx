We track dependencies for the five most recent Mastodon branches to help administrators
determine upgrade path and deployment options. Administrators should strive to run
supported releases on supported platforms to give their users the best, and most secure,
experience possible.

![Mastodon Branches](/mastodon-branches.png)

:::tip
Any dependencies listed for development versions (nightly/alpha) of Mastodon are based on
tracking the development process of the `main` branch on GitHub. These are subject to
change at any time as project requirements change.
:::

<LifecycleTable />

References: [Mastodon Security Policy](https://github.com/mastodon/mastodon/security)

:::caution
For the purposes of this document, when applied to dependencies the term "End of Life" is
standardized to mean when there are no further updates of any kind, and/or no support
available for a released version of Mastodon or its dependencies.
:::

:::danger
End of Life versions of Mastodon do not receive any features, bug fixes, and generally no
security updates except at the discretion of the development team.
:::
